# ☁️ Cloud Failure Recovery Dashboard

## 📌 Overview
This project simulates a cloud-based failure detection and recovery system using an interactive dashboard. It demonstrates how modern distributed systems monitor failures and ensure system reliability through recovery mechanisms.

## 🚀 Features
- Failure detection simulation
- Recovery strategy visualization
- Real-time system status dashboard
- Interactive UI for monitoring system behavior

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js / pyhton
- Concepts: Cloud Computing, Distributed Systems, System Monitoring

## ⚙️ How It Works
The system simulates failures in a cloud environment and triggers recovery mechanisms. The dashboard updates system status and shows how recovery strategies maintain system stability.

## 🌐 Future Scope
- Integration with AWS cloud services
- Real-time monitoring using APIs
- Automated failover and load balancing
